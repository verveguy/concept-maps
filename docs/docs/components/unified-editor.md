# UnifiedEditor

The `UnifiedEditor` component provides a unified interface that switches between graph view and text view.

## Overview

`UnifiedEditor` manages:

- View mode switching (graph/text)
- Graph view rendering
- Text view rendering
- Mode synchronization

## Features

- **Dual Views**: Seamless switching between graph and text modes
- **Bidirectional Sync**: Changes in one view reflect in the other
- **Mode Persistence**: Remembers user's preferred view mode
