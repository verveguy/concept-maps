# ConceptMapCanvas

The `ConceptMapCanvas` component is the main container for the graph view of concept maps.

## Overview

`ConceptMapCanvas` wraps React Flow and provides:

- Canvas rendering
- Zoom and pan controls
- Node and edge rendering
- Layout management
- Interaction handling

## Features

- **React Flow Integration**: Uses React Flow for graph rendering
- **Layout Algorithms**: Supports force-directed and hierarchical layouts
- **Interaction**: Drag nodes, create connections, select elements
- **Controls**: Zoom, pan, fit view
