# Getting Started

This guide will help you get started with the Concept Mapping Tool.

## Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An InstantDB account (free tier available)

## Creating Your First Map

1. **Sign Up or Log In**
   - Click the "Sign Up" button if you're new
   - Or log in with your existing account

2. **Create a New Map**
   - Click "New Map" in the sidebar
   - Give your map a name
   - Click "Create"

3. **Add Your First Concept**
   - Double-click on the canvas to create a concept
   - Type the concept name
   - Press Enter to confirm

4. **Create Relationships**
   - Click and drag from one concept to another
   - Enter the relationship label
   - Press Enter to confirm

## Editing Modes

The tool supports two editing modes:

- **Graph View**: Visual node-and-edge representation (default)
- **Text View**: Structured text representation with triple notation

Switch between modes using the toggle in the top toolbar.
