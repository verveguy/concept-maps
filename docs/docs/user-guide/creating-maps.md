# Creating Maps

A concept map is a visual representation of knowledge organized as a network of concepts connected by relationships.

## Creating a New Map

1. Click the "New Map" button in the sidebar
2. Enter a name for your map
3. Click "Create"

The map will appear in your sidebar and open in the editor.

## Map Properties

Each map has the following properties:

- **Name**: The display name of the map
- **Created By**: The user who created the map
- **Created At**: Timestamp of creation
- **Updated At**: Timestamp of last modification

## Opening a Map

Click on any map in the sidebar to open it. The map will load with all its concepts and relationships preserved.

## Deleting a Map

Right-click on a map in the sidebar and select "Delete". This action cannot be undone.

## Sharing Maps

Maps can be shared with other users. See the [Sharing Guide](/docs/user-guide/sharing) for more details.
