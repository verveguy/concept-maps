# Sharing

Maps can be shared with other users, enabling collaboration and controlled access.

## Sharing a Map

1. Open the map you want to share
2. Click the "Share" button in the toolbar
3. Enter the email address of the user you want to share with
4. Select the permission level:
   - **View**: Can view but not edit
   - **Edit**: Can view and edit
   - **Manage**: Can edit map and manage shares (invite others, update permissions, revoke shares)
5. Click "Share"

## Permission Levels

### View Permission

Users with view permission can:
- View the map
- See all concepts and relationships
- See notes and metadata
- Cannot make any changes

### Edit Permission

Users with edit permission can:
- View the map
- Create, edit, and delete concepts
- Create, edit, and delete relationships
- Edit notes and metadata
- Create perspectives
- Cannot invite others or manage shares

### Manage Permission

Users with manage permission can:
- All capabilities of Edit permission
- Invite others to share the map
- Update permission levels for existing shares
- Revoke shares and invitations
- Manage all aspects of map sharing (except deleting the map itself)

## Managing Shares

1. Open the share dialog
2. View all users who have access
3. Change permissions or remove access as needed

**Note:** Only map owners and users with Manage permission can manage shares. Users with Edit permission cannot invite others or modify share permissions.

## Map Ownership

The creator of a map is the owner and has full control:
- Can edit and delete the map
- Can manage all shares
- Cannot be removed from the map

Owners have all the capabilities of Manage permission, plus the ability to delete the map itself.

## Invitations

When you share a map, the recipient receives an invitation link. They can accept the invitation to gain access to the map.
