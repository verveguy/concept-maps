# Concept Mapping Tool Documentation

This directory contains the Docusaurus documentation site for the Concept Mapping Tool.

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm start

# Build for production
pnpm build

# Serve production build locally
pnpm serve
```

## Documentation Structure

- **User Guide** (`docs/user-guide/`): End-user documentation
- **Components** (`docs/components/`): React component API documentation
- **Architecture** (`docs/architecture/`): System design and technical architecture

## Deployment

The documentation is automatically deployed to GitHub Pages alongside the main application via GitHub Actions.

The documentation is available at: `https://verveguy.github.io/concept-maps/docs/`

The main application is available at: `https://verveguy.github.io/concept-maps/app/`
